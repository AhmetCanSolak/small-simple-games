# Tic Tac Toe

Well known Tic Tac Toe game.

Currently only off-line multi-player mode is available.

## Requirements
Python 3.5.2 at least

Pygame 1.8 at least

## How to Run the Game

```python
python3 /tic_tac_toe/main.py
```

## Keyboard Settings
- Use mouse to pick the box
- Press Q to quit :)
